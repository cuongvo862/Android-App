Instructions for running the RentalZ application
1. Extract the file
2. After extracting open the file with Visual Studio Code
3. Right-click the file and select Open in Integrated Terminal
4. Type the command "ionic serve" into the Terminal to run the application
5. The application will automatically run in your browser, now you just need to press the F12 button
6. Next in the Dimensions section select the device as Moto G4, then select Show device frame
